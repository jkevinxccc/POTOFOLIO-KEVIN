About Me

Web Developer

Haloo! Saya Kevin, siswa SMK jurusan Rekayasa Perangkat Lunak (RPL) yang memiliki minat dan passion besar di dunia pengembangan web. Sejak mengenal dunia coding, saya tertarik untuk terus belajar dan mengembangkan kemampuan saya dalam membuat website yang tidak hanya menarik secara tampilan, tetapi juga fungsional dan responsif.
Melalui berbagai proyek pribadi maupun tugas sekolah, saya telah membangun beberapa website menggunakan HTML, CSS, JavaScript, dan framework seperti Bootstrap dan Laravel. Portofolio ini saya buat untuk menampilkan hasil karya saya selama menempuh pendidikan di SMK, sekaligus sebagai bukti nyata semangat saya dalam terus berkembang di dunia teknologi.
